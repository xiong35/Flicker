# NABCD 模型分析

## N(Need)

基于 Web 平台的知识分享平台

实现一个自定义化程度更高, 更好用的学习分享平台

## A(Approach)

- 支持自定义题库
- 支持分享题库
- 支持评论题目
- 支持收藏他人发布的题库
- 支持多种题目模式
- 题目与答案均支持富文本(markdown 及 katex)

## B(Benefit)

- 能学习自己真正感兴趣的东西
- 能结识同好
- 能让学习过程更合自己心意
- 能享受更好看的界面

## C（Competitors）

现有学习软件的不足:

- 自定义程度低, 只能使用他们提供的题库
- 题目多样性差, 大多以选择题为主
- 冗余功能多, 会有广告等
- 界面较为死板, 无多主题选择
- 鲜有针对单一题目的讨论区

## D（Delivery）

- 增加更多题目类型支持
- 支持 音视频 等学习媒介
- 增加审核
