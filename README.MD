## :sparkles: Features

- 支持自定义题库
- 支持分享题库
- 支持评论题目
- 支持收藏他人发布的题库
- 支持多种题目模式
- 题目与答案均支持富文本(markdown 及 katex)
- 支持多种主题样式

## :rocket: Technologies

- 前端: React, vite, TypeScript, Scss, Material-UI,
- 后端: golang, echo, mongo, redis
- 运维: husky, docker, docker-compose, nginx, drone-ci
